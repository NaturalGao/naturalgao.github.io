重拾PHP——开篇

2019-12-08

  2. 2. 为什么要写这个系列
  3. 3. PHP基本介绍

1. 自我介绍

​ 笔者是一个还没毕业的大专生，就读”网站建设“专业。虽未毕业，已实际接触这个行业三年。在校期间帮人维护网站，接外包等…，掌握基本的前端知识，主学PHP，主要掌握的框架有Laravel和ThinkPHP。

2. 为什么要写这个系列

​ 为什么我会写这个系列呢？我相信大家都会有那么一个经历，迷茫的时候。

​ 有时候大家都”太爱学习“了，什么都想学。看到类似XXX语言，XXXX技术好吊，XXXX解决了什么，XXXX太好用了~~，是不是都有立马想去学的冲动，什么技术都想学。

​ 我最近也出现了这种情况，我常看一些JAVA，微服务，算法，数据结构相关知识，虽然一看就会的那种，但是我重新审视了下自己，是否太过贪婪？

​ 认识到了自己的这一点，我决定重新把PHP走一遍，让自己进一步掌握和了解PHP的精髓。笔者认为这种求知欲是好了，但是我们应该理智分析,学习技术应该分两种：

  1. 当下应该学的 （优先）
  2. 扩展知识面 （了解）

总结

  1. 先在自己擅长的领域站稳脚步，再扩展其它的领域。
  2. 不要太过贪婪，不要一下子就想吃成个”胖子“，消化不了，身体承受不了。
  3. 应该优先学习当下能用到的，例如工作中，自己的某个项目。其次再扩展自己的知识面。

3. PHP基本介绍

​ 学东西怎么能不知道它的用途呢？

​ PHP（“PHP: Hypertext Preprocessor”，超文本预处理器的字母缩写）是一种被广泛应用的开放源代码的多用途脚本语言，它可嵌入到 HTML中，尤其适合 web 开发。（这是官方的对PHP的定义）

​ 简单来说，PHP就是运行在服务端的解释型脚本语言.我们通常用它来开发web应用。

4. 重拾计划

​ 如果有人问我如何提高学习效率？那我一定回推荐他去看官方文档,因为这份文档最权威，最详细。

所以此次重拾计划也是根据官方的文档进行的。
